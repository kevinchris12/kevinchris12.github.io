Gemini Nano Banana Image Generator

Esta es una aplicación web para generar imágenes con una interfaz similar a ComfyUI. Permite ingresar múltiples referencias, un prompt negativo, configurar el formato de salida de la imagen y descargar la imagen generada. Esta aplicación se creo usando Google AI Studio. El modelo por defecto para generar imágenes en esta aplicación es el LLM Gemini 2.5 Flash Image usando Nano Banana.
